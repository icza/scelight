Based on your computer skills ${appName} can hide or simplify some things which you might not need or might not understand, resulting in clearer user interface and better user experience.

The Computer Skill Level setting determines what information, options, settings and controls are presented to you. Higher skill levels generally offer more details, more customization, more controls and deeper insight.

The Basic level is recommended if you are not very good with computers, and you just want ${appName} to work with minimal effort and hassle. [To lighten up]

The Normal level is recommended for the average / casual users, if you plan to use ${appName} regularly . [To follow the path of light]

The Advanced level is recommended if you have advanced computer skills and / or you want to get the most out of ${appName}. [To understand the physics of light]

The Developer level is for computer programmers / developers , this level shows you details and unlocks tools which require programming knowledge to understand. [To be the source of light]

You can change this at any time.

Note: The Developer level is disabled by default. You have to prove your developer skills to unlock it.
To unlock the Developer level:
  • Create a file "dev-skill-level-ticket.txt" in your Workspace
    • Please check under Settings >> Workspace for more information about workspace
  • Add the solution of following problem to this file
    1. Find the maximum value of an 8-byte signed integer
    2. Right Shift the value from 1. by 0x1A
    3. XOR the value from 2. with 0xCAFEBABE
    4. Get the decimal representation of the value of 3.
    5. Create the Hexadecimal SHA-256 of the value of 4.. Handle the Value from 4 as a string
